网易云课堂

网易云课堂下载教程

m3u8很容易抓，教程略，但是key只允许访问一次，抓到key即可解密

火狐浏览器

1.找到你要下载的课程

火狐浏览器播放课程

2.F12过滤key?

3.复制Key为Base64格式

找到那个16字节的请求，右键，复制响应

4.进行下载即可

将Base64复制到软件界面，即可正常下载

Chrome浏览器

1.找到你要下载的课程

Chrome浏览器打开，同时打开Fiddler进行抓包（软件基础安装使用教程略）

2.在浏览器播放

3.fiddler过滤key的请求

切换到Filters选项卡，勾选Use Filters、在下方的Request Headers中勾选Show only if URL contains，填写key?，最后点击右上角的Actions-Run Filterset now，即可过滤出所需内容

4.复制Key为Base64格式

切换到Inspectors选项卡，并选择下方区域的HexView

选中黑色部分，点击右键-Copy-Copy as Base64

5.进行下载即可

将Base64复制到软件界面，即可正常下载
